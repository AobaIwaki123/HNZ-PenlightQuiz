# 日向坂46ペンライトクイズ

2024年3月時点のメンバーで、日向坂46のペンライトの色の組み合わせを当てるクイズを作成しました。  
初めてのWebアプリということもあり、レスポンシブなデザインにはなっていません。  
ですので、プレイされる場合はパソコンでのプレイをお勧めします。

## 事前準備

- [データベースの作成手順](./docs/create-db.md)

## Quick Start

```sh
$ make init
```
